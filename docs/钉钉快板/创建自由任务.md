# 创建自由任务

调用本接口，创建一个钉钉快办自由任务（非协作空间的任务）。

### 什么是自由任务

自由任务是在钉钉快办中直接创建，但是不关联任何协作空间的任务。 ![](https://img.alicdn.com/imgextra/i3/O1CN010Hf2qH1seKcESciFx_!!6000000005791-0-tps-1530-445.jpg)

### 接口功能介绍

调用本接口，给员工小钉创建一个名为**测试任务** 的任务。接口调用成功后，员工小钉在我的任务列表查看到的信息如下图所示，有一个名为**测试任务** 的任务。 ![](https://img.alicdn.com/imgextra/i4/O1CN01mJuwTe1nqdAeG4rWT_!!6000000005141-0-tps-1676-465.jpg)
