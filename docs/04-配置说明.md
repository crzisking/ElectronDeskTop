# 配置說明文檔

## 1. 概述

`app-config.json` 是應用的**唯一外部配置文件**，放置於應用數據目錄下。
通過修改此文件，可以在不重新編譯的情況下調整所有菜單、系統鏈接、功能開關和 API 地址。

**文件位置：**
- 開發環境：`config/app-config.json`（項目根目錄）
- 生產環境：`{userData}/app-config.json`（應用自動從安裝包解壓默認配置到此處）
  - Windows: `C:\Users\<User>\AppData\Roaming\enterprise-desktop-client\app-config.json`
  - macOS: `~/Library/Application Support/enterprise-desktop-client/app-config.json`

**加載時機：** 應用啟動時由主進程讀取，通過 IPC 發送至渲染進程，存儲在 `configStore`。

## 2. 完整 TypeScript 類型定義

```typescript
// src/types/config.types.ts

export interface AppConfig {
  /** 配置文件版本號，用於升級遷移 */
  version: string

  /** 全局應用設置 */
  app: AppSettings

  /** 左側邊欄菜單配置 */
  sidebar: SidebarConfig

  /** 浮動小球配置 */
  floatingBall: FloatingBallConfig

  /** 統一平台頁面配置 */
  unifiedPlatform: UnifiedPlatformConfig

  /** AI 快捷功能配置 */
  aiQuickFunctions: AiQuickFunctionsConfig

  /** 快速聯繫功能配置 */
  quickContact: QuickContactConfig
}

// ─── 全局設置 ────────────────────────────────────────
export interface AppSettings {
  /** 界面語言 */
  language: 'zh-TW' | 'zh-CN' | 'en'
  /** 界面主題 */
  theme: 'light' | 'dark' | 'system'
  /** 啟動時最小化到浮球（跳過主窗口）*/
  startMinimized: boolean
  /** 開機自啟動 */
  launchOnStartup: boolean
}

// ─── 側邊欄 ──────────────────────────────────────────
export interface SidebarConfig {
  /** 默認是否折疊側邊欄 */
  defaultCollapsed: boolean
  /** 側邊欄菜單項列表（順序即為顯示順序）*/
  items: SidebarItem[]
}

export interface SidebarItem {
  /** 唯一 ID，對應 Vue Router 命名路由 */
  id: string
  /** 顯示名稱 */
  label: string
  /** Element Plus 圖標名或 SVG 路徑 */
  icon: string
  /** Vue Router 命名路由名（name 字段）*/
  routeName: string
  /** 是否啟用（false 則不顯示）*/
  enabled: boolean
  /** 可選：側邊欄右側 badge 文本 */
  badge?: string
}

// ─── 浮球 ────────────────────────────────────────────
export interface FloatingBallConfig {
  /** 浮球直徑（像素），默認 60 */
  size: number
  /** 浮球透明度 0.0 ~ 1.0，默認 0.9 */
  opacity: number
  /** 浮球初始位置（屏幕座標）*/
  defaultPosition: { x: number; y: number }
  /** 拖動停止後是否吸附到屏幕邊緣 */
  snapToEdge: boolean
  /** 右鍵菜單項列表 */
  quickMenu: QuickMenuItem[]
}

export interface QuickMenuItem {
  /** 唯一 ID */
  id: string
  /** 顯示名稱（separator 時可為空字符串）*/
  label: string
  /** Element Plus 圖標名（可選）*/
  icon?: string
  /** 點擊後執行的操作 */
  action: QuickMenuAction
  /** 是否啟用 */
  enabled: boolean
  /** 是否在此項後面渲染分隔線 */
  separator?: boolean
}

/** 快捷菜單操作類型（聯合類型） */
export type QuickMenuAction =
  | { type: 'show-main-window' }                            // 顯示主窗口
  | { type: 'navigate'; routeName: string }                  // 導航到指定頁面
  | { type: 'open-url'; url: string; target: 'browser' | 'iframe' } // 打開 URL
  | { type: 'quit-app' }                                    // 退出應用

// ─── 統一平台 ─────────────────────────────────────────
export interface UnifiedPlatformConfig {
  /** 公司內部系統列表 */
  systems: SystemLink[]
}

export interface SystemLink {
  /** 唯一 ID */
  id: string
  /** 系統名稱 */
  name: string
  /** 系統描述 */
  description: string
  /** 系統 URL */
  url: string
  /** 系統圖標 URL（可選，無則顯示默認圖標）*/
  iconUrl?: string
  /** 打開方式 */
  openMode: 'iframe' | 'external-browser'
  /** 是否啟用 SSO 自動登錄 */
  ssoEnabled: boolean
  /** SSO Token 注入的 URL 查詢參數名 */
  ssoTokenParam?: string
}

// ─── AI 快捷功能 ──────────────────────────────────────
export interface AiQuickFunctionsConfig {
  /** AI 後端 API 基礎地址 */
  apiBaseUrl: string
  /** 啟用的 AI 工具（決定 Tab 是否顯示）*/
  enabledTools: Array<'text-processor' | 'summarizer' | 'qa'>
  /** 輸入文本最大字符數限制 */
  maxInputLength: number
}

// ─── 快速聯繫 ─────────────────────────────────────────
export interface QuickContactConfig {
  /** 聯繫人搜索 API 地址 */
  searchApiEndpoint: string
  /** 發送郵件 API 地址 */
  emailApiEndpoint: string
  /** 搜索結果最大條數 */
  maxSearchResults: number
}
```

## 3. 字段說明

### 3.1 `app` 全局設置

| 字段 | 類型 | 必填 | 默認值 | 說明 |
|------|------|------|--------|------|
| `language` | enum | 是 | `"zh-TW"` | 界面語言 |
| `theme` | enum | 是 | `"system"` | `"system"` 跟隨 OS 暗色模式 |
| `startMinimized` | boolean | 是 | `false` | `true` 時啟動不顯示主窗口，直接顯示浮球 |
| `launchOnStartup` | boolean | 是 | `false` | 是否加入系統開機自啟動 |

### 3.2 `sidebar.items` 側邊欄項

| 字段 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `id` | string | 是 | 唯一標識，建議與 routeName 相同 |
| `label` | string | 是 | 顯示文字 |
| `icon` | string | 是 | Element Plus 圖標名，如 `"Grid"`, `"MagicStick"`, `"User"` |
| `routeName` | string | 是 | Vue Router 路由 name，點擊時 `router.push({ name })` |
| `enabled` | boolean | 是 | `false` 則該菜單項不渲染 |
| `badge` | string | 否 | 側邊欄右側徽標，如 `"New"`, `"99+"` |

### 3.3 `floatingBall.quickMenu` 浮球快捷菜單

| 字段 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `id` | string | 是 | 唯一標識 |
| `label` | string | 是 | 菜單文字（`separator: true` 時可為空）|
| `icon` | string | 否 | Element Plus 圖標名 |
| `action.type` | enum | 是 | 見下方 action 類型說明 |
| `enabled` | boolean | 是 | `false` 則不渲染此項 |
| `separator` | boolean | 否 | `true` 時在此項後渲染分隔線 |

**action 類型說明：**

| `action.type` | 額外字段 | 行為 |
|---------------|----------|------|
| `show-main-window` | 無 | 顯示主窗口，隱藏浮球 |
| `navigate` | `routeName: string` | 顯示主窗口並導航至指定路由 |
| `open-url` | `url: string`, `target: 'browser'\|'iframe'` | 打開外部 URL |
| `quit-app` | 無 | 退出整個應用 |

### 3.4 `unifiedPlatform.systems` 系統鏈接

| 字段 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `id` | string | 是 | 唯一標識 |
| `name` | string | 是 | 系統名稱 |
| `description` | string | 是 | 系統描述 |
| `url` | string | 是 | 系統訪問地址 |
| `iconUrl` | string | 否 | 系統圖標，支持 URL 或相對路徑 |
| `openMode` | enum | 是 | `"iframe"` 在窗口內嵌入；`"external-browser"` 在系統瀏覽器打開 |
| `ssoEnabled` | boolean | 是 | `true` 時自動將 Auth Token 注入 URL |
| `ssoTokenParam` | string | 否 | Token 注入的參數名，`ssoEnabled: true` 時必填 |

## 5. 配置加載流程

```
應用啟動
    │
    ▼
Main 進程讀取 app-config.json
    │
    ├─ 文件存在 → 解析 JSON → 版本校驗 → 與默認值 deepMerge
    │
    └─ 文件不存在 → 從安裝包複製 default-config.json 到 userData → 讀取
    │
    ▼
Renderer 請求 IPC: CONFIG_READ
    │
    ▼
configStore.loadConfig()
    │
    ├─ 校驗必填字段
    ├─ 存儲到 configStore.appConfig
    └─ 通知各 Store 初始化完成
```

## 6. 配置熱更新（可選）

如需支持管理員更新配置文件後無需重啟應用：

1. 主進程使用 `fs.watchFile()` 監聽 `app-config.json` 變更
2. 文件變更後重新讀取並驗證
3. 通過 `mainWindow.webContents.send(IpcChannels.PUSH_CONFIG_CHANGED, newConfig)` 推送到渲染進程
4. 渲染進程監聽此事件並更新 `configStore`
5. 組件通過響應式數據自動重新渲染

> 注意：熱更新不會重新創建浮球窗口，浮球的外觀變化（size/opacity）需重啟後生效。
